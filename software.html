Displaying code in your webpage with css

Check out this page where the author nicely describes what the issues are in displaying programs in html pages and how you could overcome the problem : Code Boxes with pre and CSS


Here is the gist of it. Use the 'pre' tag to embed your code and your job is done. If you want something more fancy you can modify its behavior with the css. Below is the css code they suggest on the above page. 

                
/* Code Samples */
pre {
 white-space: pre-wrap;
 white-space: -moz-pre-wrap;
 white-space: -o-pre-wrap;
 background: #faf8f0;
}
#content #pre-stylish code {
 display: block;
 padding: 0.5em 1em;
 border: 1px solid #bebab0;
}
